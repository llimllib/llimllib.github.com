Photo Gallery

I am currently checking out the photo galleries that exist for pyblosxom. I like the idea given by Roberto de Almeida, who uses RDF embedded in JPEG comment fields to store image metadata. I would like it to be able to display galleries in a much more complete way than it seems to allow, as well as allow comments to files, so I'll be looking at extending it. That would give me a chance to learn some XML and tinker with the JPEG file format, so I think I'll give it a spin.

It seems that the only other photo gallery software is this plugin; I'll give it a try. Since I've been thinking about this for a while, though, I'm willing to bet that I'll try my hand at writing my own.

UPDATE: I'm going to check out FotoNotes for image metadata as well.
